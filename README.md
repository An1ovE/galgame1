# 🌸 Sakura Romance Engine

樱花与迷雾交汇之章 · AVG 动态数值决策系统

一个以大学校园为背景的恋爱模拟视觉小说游戏，采用粉白玻璃拟物化设计风格。

## 🎮 游戏简介

玩家扮演京传机械系大二学生，在跨校大创项目中与京化大 502 宿舍的四位女生相识。游戏包含：

- **第一章**：初次相遇（四位女主角的独立邂逅）
- **第二章**：五零二的邀请（跨校大创组队）
- **第三章**：胡同里的晚宴（项目庆功与情感积累）
- **第四章**：交汇与边界（好感度路由判定，进入专属路线）
- **第五章**：平行的交点（三阶段渐进式暧昧，HE/BE 分支）
- **第六章**：终章·岁月与你（毕业分流、年度模拟、婚姻大考）

## 🚀 运行方式

直接用浏览器打开 `index.html` 即可运行，无需任何服务器或依赖。

也可部署到 GitHub Pages：
1. Fork 本仓库
2. Settings → Pages → Source 选择 `main` 分支
3. 访问 `https://<你的用户名>.github.io/sakura-engine/`
